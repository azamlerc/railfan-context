# andrewzc.net — Page Reference

This is a reference to all list pages on andrewzc.net and their keys. Use it to identify which list to search or update when Andrew gives a command.

Page keys are used in API calls (`/entities/:list/:key`, `/pages/:key`) and in links (`page.html?id=LIST`).

## Page size values

- `small` — compact grid, suitable for long lists of flags/icons
- `medium` — standard grid, most lists
- `large` — full-width rows with more detail

## Page type values

- `place` — physical locations
- `entity` — things (cars, albums, etc.)
- `city` — city-scoped list
- `country` — country-scoped list
- `state` — state/province list
